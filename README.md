--

## Quick start
1. `git clone https://github.com/mmdsharifi/Bootstrap4-RTL.git`
2. `cd Bootstrap4-RTL`
3. `[sudo] npm install`
4. `gulp` and enjoy !

## Install with bower
`bower install bootstrap4-rtl`
